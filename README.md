# GreenBreath-Indoor-Air-Balance-ML
ML-based web application for predicting indoor air quality and CO₂ balance using Linear Regression and synthetic environmental data.

# Overview
GreenBreath is an ML-based web application that predicts indoor air quality and analyzes whether indoor plants can balance CO₂ levels in closed rooms using Linear Regression and scientific formulas. The system evaluates room size, number of people, number of plants, and plant type to estimate indoor air conditions.
# Features
The project predicts indoor CO₂ levels, calculates Air Quality Score, estimates plant effectiveness and required plant count, classifies room condition as Good, Moderate, or Critical, and visualizes CO₂ variation over time using a 4-hour simulation graph.
# Technologies Used
The project is developed using HTML5, CSS3, and JavaScript, while Linear Regression and Batch Gradient Descent are used for Machine Learning implementation.
# Machine Learning Concepts
The project uses Supervised Learning concepts such as Linear Regression, Gradient Descent, and Synthetic Dataset Generation for predicting indoor air quality.
# Output
The system generates outputs including final CO₂ concentration (ppm), Air Quality Score, plant effectiveness, estimated plants required, and overall room air quality status.
# Conclusion
GreenBreath combines Machine Learning and environmental science to analyze indoor air quality and demonstrate the role of indoor plants in improving air balance within closed environments.
